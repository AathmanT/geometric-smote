geometric-smote
===============

Implementation of the G-SMOTE algorithm. Fully compatible with
the Scikit-Learn and Imbalanced-Learn API.

Prerequisites
=============
- NumPy
- Scikit-Learn
- Imbalanced-Learn

Installation
============

Clone the repository and activate a Python 3 virtual environment.
Then run the following commands:

.. code:: shell

    cd gsmote
    pip install .
